\section{\large 1. Problemas}
\subsection{Problema 1}
Crear una aplicación (Utlizando sockets tipo de flujo como datagrama y TCP) que desde la máquina cliente solicite a un servidor consultas sobre la base de datos (Universidad.xsl). Normalice a la 4F respectivamente  la base de datos proporcionada. La aplicación debe consultar,
modficar, eliminar o actualizar los registros de la base de datos respectivamente desde el cliente. Para el desarrolo de este problema, puede utilizar: \textbf{50 pts.}

\begin{itemize}
    \item Un gestor de base de datos (My SQL, Postgress, entre otros.)
    \item Gestión de la base de datos mediante archivos de texto.
\end{itemize}

\textbf{Nota:} Insertar todos los registros que vienen en el archivo Universidad.xls para realizar las consultas.

\subsection{Solucion}

Para poder alzar la base de datos por medio de archivos de texto, lo principal es normalizar la base de datos del archivo "Universidad.xsl", posterior a esto debemos 
registrar los datos que contiene originalmente nuestra base, a la nueva base de datos normalizada que se muestra a continuación:
